What is HTML?

HTML (HyperText Markup Language) is the fundamental language for structuring web pages. It defines the content and layout of a webpage using elements such as headings, paragraphs, links, and images.

Basic HTML Structure

Every HTML document has a standard structure. Here’s an example:

            <!DOCTYPE html>
            <html>
            <head>
                <title>My First Webpage</title>
            </head>
            <body>
                <h1>Welcome to HTML!</h1>
                <p>This is a simple webpage.</p>
            </body>
            </html>

Common HTML Elements

  • <h1> - <h6>: Headings
  • <p>: Paragraphs
  • <a href="URL">: Links
  • <img src="image.jpg" alt="Description">: Images
  • <ul> & <ol>: Lists
